Download the app when "not compatible"

Occasionally, the Dazzl application may appear to be incompatible in the Google Play Store for some Android models.

You can always install the application, using the following link:

https://my.dazzl.tv/bin/android/dazzl-mobile.apk

 

If you need help to instal the APK file, you can use this tutorial, explaining how to enable the apk installation :

https://www.wikihow.tech/Install-APK-Files-on-Android#Enabling_APK_Installations_sub
